Creating a table in MYSQL with Node.js

1. We created a "MERN" file to keep all our files together

2. Opened a terminal from the MERN file and used 'gedit filename' to start testing Node and npm

3. started by creating a demo_createDB.js file, creating a database called mydb

4. Created a demo_create_table.js file to create a table called "customers" in mydb

5. Created demo_db_insert.js file to insert a single customer into the list

6. Finally we created a demo_db_select.js file to and display the customers in the table

NPM Install

1. Now do "sudo apt install npm"

2. In order to install anything through npm all you need to do is "npm install (file)"
