GitHub uploading

Josh S. / May 16 2019

Starting git in folder

1. git install

2. git init (on terminal in current file you want to upload)

3. git config --global user.name "Your Username" and git config --global user.email "Your Email"

4. git status

5. touch .gitignore

6. git add -A

7. git commit -m "short, but thorough"

8. git remote add origin (GitHub repo URL)

9. git pull origin master - this pulls any changes that have been made in repo

10. git push origin master - pushes commit to the repo

helpful

git diff - allows you to see the difference in code between your most recent commit and last commit

git remote set-url origin (github repo url) - allows you to change youre origin remote
