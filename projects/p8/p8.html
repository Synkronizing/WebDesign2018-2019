MongoDB RegEx

Creating database

db.dogs.find({$and:[{age:{$lte:8}},{name:/^[TS].*/}]},{_id:0,name:1,age:1,tricks:1})

This mongo code finds all dogs with a name that starts with a T or an S through regular expressions or regex and are 8 or younger

The regex begin when "/" Then "^" is 'begins with' [TS] means it can be either T or S and .* means there can be 0 or more characters afterwards
