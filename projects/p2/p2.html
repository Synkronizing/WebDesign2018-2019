Installing MySQL and more!

MySQL

1. Make sure you have apache2, if not do "sudo apt install apache2"

2. Then do a "sudo apt install mysql-server"

3. Now do "sudo mysql_secure_installation" make sure to make a password you remember or just do 'toor'

4. Finally enter "sudo mysql" then you will enter "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'toor';" to be able to enter mysql correctly("mysql -u root -p")

MyPHPAdmin Install

Do this after downloading MySQL

1. do sudo apt install phpmyadmin

2. make sure to hit tab when hovering over apache2 (it should have a "*" in the box)

3. Click "yes"

4. type your password in for MySQL
